BTC-Pro buy/sell app with Rakz and Stakz tokens
# Micro Holdings — BTC-Providence (BTC-Pro)

Bitcoin Providence (BTC-Pro) is the **bridge coin** used to buy and sell the algorithmic coins:

- BTC-Pearl (BTC-P)
- BTC-Black (BTC-B)
- BTC-Platinum (BTC-Pl)

### How it works
- BTC-Pro splits into **Rakz Token** (for buying) and **Stakz Token** (for selling).
- Both are Ethereum-based ERC-20 tokens.
- Used to purchase the algorithmic coins on the Grokchain.

### Tokens in this repo
- BTCProvidence.sol — Main bridge coin
- RakzToken.sol — Buy token
- StakzToken.sol — Sell token
